一个简单的从slnx生成compile_commands.json的工具（开发中，实用性未知）

使用方法

```
Usage: Slnx2CompileCommand [--help] [--version] [--outpath VAR] file

Positional arguments:
  file           Path to slnx file

Optional arguments:
  -h, --help     shows help message and exits
  -v, --version  prints version information and exits
  --outpath      Specify the output directory [nargs=0..1] [default: "."]
```
